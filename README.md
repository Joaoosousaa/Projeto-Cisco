# Projeto-Cisco
Desenvolvi uma infraestrutura no Cisco Packet Tracer de uma empresa fictícia. Criei VLANs para segmentação lógica de departamentos e Router-on-a-Stick para o roteamento Inter-VLAN. A interligação entre Matriz e Filial é feita via VPN Site-to-Site, utilizando criptografia para garantir a proteção dos dados.
